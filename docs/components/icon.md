# Icon

## Example

下面的标签 `<mwe-icon name="success"/>` 直接写在Markdown里，可以运行demo效果.

<mwe-icon name="success"/>

当你想要只看demo源码的时候，可以用下面的方式直接写在Markdown中，这样就可以查看demo的代码部分。

``` js
<mwe-icon name="success"/>
```
